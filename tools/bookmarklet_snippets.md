- This one always attempts: **can shoot → teleport behind lowest-health target → shoot → instant retreat to safest known walkable point**.
- While reloading, it repeatedly teleports to safety (fewest nearby bullets/players) using a rebuilt walkable teleport map so it avoids walls.
- Press **P** to pause/resume after activation.
